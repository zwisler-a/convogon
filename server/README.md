# conVogon

## Features
- Login
- Spieler hinzufügen, einsehen, bearbeiten
    - OT Infos inkl. Unterkunft und Zahlungsstatus
    - IT Infos
- Admin Spieler Verwaltung
    - Einsehen
    - Zuordnung Unterkunft
    - Zahlung bestätigen
- Spieler vor Ort einchecken (z.B. via QR-Code)
- Infoseite / FAQ
- Optional: Newspage
